# AI-Driven Agile Framework

An intelligent project management platform that combines **Artificial Intelligence (AI)** and **Agile methodologies** to identify project risks, detect bottlenecks, and support data-driven decision-making throughout the software development lifecycle.

## Overview

The AI-Driven Agile Framework continuously analyzes project activities and team progress to predict potential risks before they impact delivery timelines. The platform helps project managers and development teams improve productivity, reduce delays, and enhance project success rates.

## Key Features

* Real-time project monitoring
* AI-powered risk prediction
* Bottleneck and delay detection
* Automated mitigation recommendations
* Agile workflow support
* Interactive dashboard for project insights
* Website and project analysis tools
* Machine Learning-based bug and risk assessment

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Driven-Agile-Framework.git
cd AI-Driven-Agile-Framework
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Usage

1. Start the backend server.
2. Launch the frontend application.
3. Upload project data or enter project details.
4. Analyze project health and risks.
5. Review AI-generated recommendations and mitigation strategies.
